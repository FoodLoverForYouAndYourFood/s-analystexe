# Анализ кастдева - Matcher

## Jobs To Be Done

### 1. Массовый скрининг
**Проблема:** Откликнулся 200 раз, ничего не пришло. Тратит 4-5 минут на каждую вакансию.
**Работа:** Отфильтровать 200 вакансий до 10-15 релевантных быстро.

### 2. Профиль прокачки (!!!ГЛАВНОЕ!!!)
**Проблема:** "Не хватило стека по архитектуре в Озон"
**Работа:** Взять 10-15 вакансий → понять какой стек требуется на рынке → сформировать список что учить.

### 3. Видеть скрытое
**Проблема:** "Краткое описание врет", "вчитываться в каждый нюанс", "в описании написан другой опыт"
**Работа:** Детектить несоответствия и скрытые требования.

### 4. Фильтр по типу компании
**Проблема:** "Рассматриваешь там где мультипродуктность, чтобы не уходить из конторы"
**Работа:** Фильтр: продуктовые, мультипродуктовые, домен (логистика/финтех).

### 5. Тип оформления
**Проблема:** "Тип оформления важен"
**Работа:** Показывать штат/ИП/самозанятость.

### 6. Спасти от дебилов
**Проблема:** "Спасу от дебилов в описании"
**Работа:** Red flags detector - плохо написанные вакансии.

---

## Решение для Matcher

### Текущая версия (MVP)
✅ Проверить 1 вакансию → score + рекомендация
❌ Долго (4-5 минут проще прочитать самому)
❌ Не решает главную боль - профиль прокачки

### Новая версия (V2)

#### Feature 1: Batch Analysis
- Загрузить 10-15 вакансий разом (не по одной)
- Массовый анализ за 1 запуос
- Краткая таблица: вакансия | score | главный gap

#### Feature 2: Gap Analysis (КЛЮЧЕВОЕ)
- Частота требований по стеку (Python: 12/15, Kafka: 8/15)
- Что есть в резюме vs что требуется
- Приоритизированный список "что учить"
- Roadmap прокачки на 3-6 месяцев

#### Feature 3: Smart Filters
- Тип компании: продуктовая / мультипродуктовая / аутсорс
- Домен: финтех / логистика / e-commerce
- Тип найма: штат / ИП / самозанятость / не указан
- Red flags: "молодая команда", "быстрорастущий стартап", "работа в режиме цейтнота"

#### Feature 4: Детектор несоответствий
- Краткое описание vs полное описание
- Заявленный опыт vs реальные требования
- Скрытые требования ("знание домена обязательно")
